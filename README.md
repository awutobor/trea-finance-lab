# Trea Finance Lab - 虚拟股票交易模拟器

一个基于命令行的虚拟股票交易模拟器，让用户使用10万虚拟本金体验真实股票交易。

## 功能特点

- 📈 **实时股价查询** - 使用 yfinance 获取真实股票数据
- 💰 **虚拟交易** - 买入/卖出股票，体验真实交易流程
- 📊 **资产管理** - 实时计算持仓市值和盈亏
- 💾 **数据持久化** - 使用 JSON 保存数据，重启后数据不丢失
- 🎯 **低手续费** - 模拟真实交易手续费（0.1%）

## 安装依赖

```bash
pip3 install -r requirements.txt
```

## 使用方法

### 启动程序

```bash
python3 main.py
```

### 可用命令

| 命令 | 说明 | 示例 |
|------|------|------|
| `price <股票代码>` | 查询股票价格 | `price AAPL` |
| `buy <代码> <数量>` | 买入股票 | `buy AAPL 10` |
| `sell <代码> <数量>` | 卖出股票 | `sell AAPL 5` |
| `portfolio` | 查看资产组合 | `portfolio` |
| `history` | 查看交易历史 | `history` |
| `reset` | 重置账户 | `reset` |
| `help` | 显示帮助 | `help` |
| `quit/exit` | 退出程序 | `quit` |

## 文件说明

- `main.py` - 主程序，处理用户交互
- `simulator.py` - 交易模拟器核心逻辑
- `data_fetcher.py` - 股票数据获取模块
- `portfolio.json` - 用户持仓和交易记录数据文件
- `requirements.txt` - 项目依赖

## 注意事项

1. **API 限制**: yfinance 有请求频率限制，频繁查询可能会被暂时限制
2. **数据延迟**: 股价数据可能有15分钟延迟，仅供模拟使用
3. **虚拟资金**: 所有资金均为虚拟，不涉及真实交易
4. **数据安全**: `portfolio.json` 包含你的交易数据，请妥善保管

## 支持的股票
